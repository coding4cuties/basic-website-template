# Basic Website Template with Next.js and Github pages

This site is built off the [Next.js GitHub Page static site template](https://github.com/nextjs/deploy-github-pages).

1. run `npm i` to install packages
2. Run `npm run dev` to start the server on your `localhost:3000`
3. Enjoy!
